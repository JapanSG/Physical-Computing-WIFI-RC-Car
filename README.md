# Physical-Computing-WIFI-RC-Car
A WIFI Remote Control car project
  โปรเจกต์ “WIFI Remote Control Car” เป็นการประยุกต์ใช้บอร์ด Arduino IDE ร่วมกับ ESP32 ให้ควบคุมมอเตอร์และServo เพื่อควบคุมการเคลื่อนที่ของรถจากอุปกรณ์ภายนอก เช่น สมาร์ตโฟนหรือคอมพิวเตอร์ ผ่านเว็บไซต์โดยระบบสามารถรับคำสั่งให้รถเคลื่อนที่ไปข้างหน้า ถอยหลัง เลี้ยวซ้าย และเลี้ยวขวาได้อย่างอิสระ รวมถึงสามารถหยุดการเคลื่อนที่ได้ตามคำสั่งจากผู้ควบคุม
