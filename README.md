# Physical-Computing-WIFI-RC-Car
A WIFI Remote Control car project

  โปรเจกต์ “WIFI Remote Control Car” เป็นการประยุกต์ใช้โปรแกรม Arduino IDE ร่วมกับ ESP32 ให้ควบคุมมอเตอร์และเซอร์โว เพื่อควบคุมการเคลื่อนที่ของรถจากอุปกรณ์ภายนอก เช่น สมาร์ตโฟนหรือคอมพิวเตอร์ ผ่านเว็บไซต์โดยระบบสามารถรับคำสั่งให้รถเคลื่อนที่ไปข้างหน้า ถอยหลัง เลี้ยวซ้าย และเลี้ยวขวาได้อย่างอิสระ รวมถึงสามารถหยุดการเคลื่อนที่ได้ตามคำสั่งจากผู้ควบคุม 
  
  จุดประสงค์ในการทำโปรเจคของพวกเราในครั้งนี้คือ ความท้าทายในการทำในสิ่งที่ไม่เคยได้ลองทำ หาประสบการณ์ เพิ่มความรู้ความสามารถให้กับตัวเอง
