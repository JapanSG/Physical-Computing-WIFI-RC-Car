Abstract

WIFI-RC-Car เป็นโปรเจกค์รถบังคับโดย WIFI ซื่งใช้ ESP32 เป็น MCU หลักในการควบคุมรถ และเขียนโคด Arduino ลง ESP32 รถบังคับสามารถขับได้ปกติ(เดินหน้า, ถอยหลัง, เลี้ยวซ้าย, เลี้ยวขวา) โดยควบคุมผ่าน WebBrowser สามารถใช้ Keyboard หรือ ปุ่ม บน Website บังคับได้

Experience

ตัวโปรเจ็คนี้เริ่มทำตั้งแต่สัปดาห์ที่ 2 ของเทอมนี้ ซึ่งตอนแรกได้วางแผนให้ตัวรถมี feature มากกว่านี้ เช่น ความเร่งที่ปรับได้ของรถ, เบรครถ แต่เนื่องจากเวลาที่จำกัด กับโปรเจ็ควิชาอื่นทำให้ต้องตัดทิ้ง ตัวงานกว่าจะได้ ทำจริงก็หลังจากเริ่มไปแล้วเกือบ 2 สัปดาห์ เพราะในกลุ่มไม่มีใครเคยใช้ microcontroller มาก่อนและความรู้ ด้านนี้จำกัดมากๆ จึงเสียเวลานานในการศึกษาเตรียมตัว

กระบวนการทำงานของกลุ่มจะทำเป็นขั้นเล็ก ๆ และเป็นส่วน เช่น สัปดาห์นี้ทดสอบ motor สัปดาห์ต่อไปทดสอบ Wifi แล้วทำอย่างนี้เรื่อย ๆ จนสัปดาห์สุดท้ายที่ส่วนแต่ ละส่วนใช้งานแยกกันได้แต่ยังไม่ได้ต่อร่วมกัน

อุปสรรคที่พบเจอหลัก ๆ คือโปรเจ็คอื่น มาชนกับโปรเจ็คนี้ ทำให้มีเวลาน้อยลง นอกจากนี้ยังมีปัญหาด้านขาดความรู้ด้าน Electronics ทำให้มีหลายครั้งที่ซื้อ อุปกรณ์มาผิดแล้วต้องไปซื้อใหม่ หรือมีบางครั้งที่วงจรไม่ทำงานเพราะต่อผิด
