Abstract

RC-Car-WiFi is a student project implementing a remote-controlled car using an ESP32 microcontroller programmed with the Arduino IDE. The car hosts a simple WiFi-based control API and can be driven from any web browser. Controls include forward, back, left, right and stop. The website demonstrates the hardware, shows photos from multiple angles, contains the Arduino source code and embeds the project presentation video.
